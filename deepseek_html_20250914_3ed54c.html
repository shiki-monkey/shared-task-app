Firebase Setup Required

  1. Go to Firebase Console
  2. Create a new project or select an existing one
  3. Enable Authentication: Build → Authentication → Sign-in method → Email/Password
  4. Create Realtime Database: Build → Realtime Database → Create Database
  5. Get your config: Settings (gear icon) → Project settings → Your apps → Web app
  6. Replace the firebaseConfig values below with your own
